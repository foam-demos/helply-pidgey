# Support Platform Web

Next.js 14 frontend for the Helply support platform. Provides inbox UI, analytics dashboards, and agent configuration. Uses React Server Components and Tailwind CSS for styling.

**Tech Stack:** Next.js 14, React 18, TypeScript 5.3, Tailwind CSS 3.4, Vercel deployment

**Run locally:** `npm install && npm run dev` (starts on localhost:3000)
